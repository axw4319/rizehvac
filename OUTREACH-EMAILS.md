## Send sequencing notes

- Personalize the first sentence with something verifiable from each contractor's website (their About page, latest blog post, recent project photo, etc.) — generic hooks get filtered as spam.
- Each email's "rank driver" call-out is honest: tied to the actual `bestFor` field in `dallas-contractors.ts`. Use it as the anchor.
- If sending more than 1-2/day from aaron@rizehvac.com, watch the deliverability score — Postmark or SendGrid for batch (Aaron has both via ProCloser).
- Track opens via plain UTM-tagged link (`?utm_source=outreach&utm_campaign=claim-profile&utm_content=rank-1`). Don't use a tracking pixel — those tank deliverability.
- Save replies in a Google Sheet keyed by contractor name. Aaron's `grace-brains` repo has a similar pattern from ProCloser outreach.
